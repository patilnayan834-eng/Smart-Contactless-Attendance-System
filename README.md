# Smart-Contactless-Attendance-System


## 📌 Overview

The **Smart Contactless Attendance System** is an intelligent solution designed to automate attendance tracking without physical contact. It improves efficiency, accuracy, and hygiene by utilizing modern technologies such as facial recognition, QR codes, or RFID-based identification.

---

## 🚀 Features

* Contactless attendance marking
* Real-time attendance monitoring
* Automated data recording and reporting
* Secure and reliable data management
* Fast and efficient user identification
* Clean and user-friendly interface

---

## 🛠️ Technologies Used

* Python
* OpenCV
* Flask
* SQLite

## ▶️ Usage

The system allows users to register and mark attendance using a contactless method. Attendance records are automatically stored and can be accessed for tracking and analysis.
